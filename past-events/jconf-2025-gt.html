COMUNIDAD JAVA DE GUATEMALA

Organizacion de eventos

JCONF 2025

  • Categoria: Tech-JAVA-JVM
  • Aforo: 200+
  • Website: https://2025.jconf.gt
JConf Guatemala es un evento anual cuyo objetivo es crear un centro para líderes, entusiastas y desarrolladores de Java en Guatemala, Centroamérica y el mundo :)

¿Por qué es importante?

El evento es promovido por el Grupo de Usuarios de Java de Guatemala -GuateJUG-, que realiza reuniones mensuales abiertas a todos, brindando un espacio de discusión en temas relacionados con Java (como JVM). El principal objetivo del evento es promover Java como plataforma fomentando el conocimiento abierto como otros espacios ya existentes en ciudades del mundo.

Representando a Guatemala

JConf Guatemala es la conferencia de Java más antigua de Centroamérica (desde 2011), siendo el evento de Java más exitoso en la región (hasta donde sabemos :)).
